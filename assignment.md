# Insertion Sort Project

## Question 1
Write the steps of the Insertion Sort algorithm for the array  
[22,27,16,2,18,6].

### Answer
Initial state:
[22,27,16,2,18,6]

Step 1:
[16,22,27,2,18,6]

Step 2:
[2,16,22,27,18,6]

Step 3:
[2,16,18,22,27,6]

Step 4:
[2,6,16,18,22,27]

## Question 3
After the array is sorted, which of the following cases does the number 18 fall into?

- Average case: The searched element is in the middle  
- Worst case: The searched element is at the end  
- Best case: The searched element is at the beginning  

### Answer
Since the number 18 is located in the middle of the array, it falls into the **Average Case**.
